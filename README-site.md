# accord.express — site

This branch contains a minimal static website for accord.express.

Files included:
- index.html — landing page
- styles.css — site styles
- script.js — small interactions (mobile nav, smooth scroll)

How to run locally:
- Clone the repo
- Checkout the website branch:
  git checkout website/initial
- Open `index.html` in your browser. No server required for this static prototype.

Publishing
- You can enable GitHub Pages on the `website/initial` branch or move files to `gh-pages` or the default branch for automatic publishing.

If you'd like, I can add a simple deploy workflow or wire up a custom domain.
